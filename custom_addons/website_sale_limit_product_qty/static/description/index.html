Website Sale Limit Product Quantity

Limit Max. & Min. Ordered Qty. per Customer

Few Words About TheWebsite Sale Limit Product Quantity

This module will help you to limit/bound the minimum and maximum quantity of a product that can be ordered in each order from website by a customer.

Main Features
  • Enable products order limit for specific websites.
  • Set different product specific min and max limits.
  • Even you can set different limits for variants of same product.
  • Limit is also validated based on last ordered quantity.
  • The max order quantity restriction for a product is managed by using a date range, after this duration the restriction will be auto revoked and customer can order more quantities of that product.
Work Flow: How The Website Sale Limit Product Quantity Works
Enable Product Min-Max limit for a Website
Set the Min-Max limit for a product
Min quantity limit applied for the product
Max quantity limit applied for the product
Error message showing when already added all allowed quantity in cart
Double check the cart on checkout
Double check the cart on checkout
Product limit added for a customer
Quantity limit validation for new order based on previous orders
